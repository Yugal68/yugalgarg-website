## How to Make Changes

1. Edit files locally in `/Users/yugalgarg/Desktop/PreProd/WebsiteYugal`
2. Test with `npm run dev` at http://localhost:3000
3. Deploy with:
   ```bash
   git add .
   git commit -m "Description of changes"
   git push
   ```
4. Vercel auto-deploys in 1-2 minutes
